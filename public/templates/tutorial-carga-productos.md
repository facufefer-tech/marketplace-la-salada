# Tutorial rápido para feriantes (demo)

1. Descargá una plantilla CSV (`productos-general.csv`, `productos-remeras.csv` o `productos-calzado.csv`).
2. Abrila en Google Sheets o Excel.
3. Completá cada fila con tus datos reales:
   - `nombre`, `descripcion`, `precio`, `categoria`, `talle`, `color`, `stock`, `foto_url`.
4. Guardá/exportá en formato CSV.
5. Subí el archivo al panel `/dashboard/productos` o cargá producto por producto.
6. Si querés, usá el chat IA del panel para crear productos desde texto natural.

## Recomendaciones de fotos

- Fondo limpio.
- Buena luz natural.
- 1 foto frontal + 1 detalle + 1 uso real.
- Podés usar imágenes de celular.
